# vuGetMailLog()

## Description
This is the legacy `vuMail` function name for `vuLogGetFile()`.

Preferred function name:

- [vuLogGetFile()](vuLogGetFile.md)

It behaves the same way as `vuLogGetFile()` and returns the current runtime log file path.
