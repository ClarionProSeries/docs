# vuGlobalsSetCodePage()

## Description
This function sets the current runtime code page using the globals-oriented naming style.

Preferred simpler pair:

- [vuSetCodePage()](vuSetCodePage.md)
- [vuGetCodePage()](vuGetCodePage.md)

It updates the same live runtime value used by `vuSetCodePage()`.
