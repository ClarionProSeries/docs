# vuGetCodePage()

## Purpose
Returns the current runtime code page value.

## Parameters
- None.

## Return value
Returns the currently active numeric code page.

## Notes
- This reads the current in-memory runtime value.
- Use [vuSetCodePage()](vuSetCodePage.md) to change it.
- The globals-oriented aliases are [vuGlobalsGetCodePage()](vuGlobalsGetCodePage.md) and [vuGlobalsSetCodePage()](vuGlobalsSetCodePage.md).
