# vuGlobalsGetEmailFolder()

## Purpose
Returns the current runtime email folder value used by related save/export operations.

## Parameters
- None.

## Return value
Returns the current folder path stored in live runtime memory.

## Notes
- This reads the current in-memory value.
- Use [vuGlobalsSetEmailFolder()](vuGlobalsSetEmailFolder.md) to change it.
