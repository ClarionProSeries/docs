# vuGlobalsGetCodePage()

## Description
This function returns the current runtime code page using the globals-oriented naming style.

Preferred simpler pair:

- [vuGetCodePage()](vuGetCodePage.md)
- [vuSetCodePage()](vuSetCodePage.md)

It returns the same live runtime value used by `vuGetCodePage()`.
