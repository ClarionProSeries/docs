# vuSetMailLog()

## Description
This is the legacy `vuMail` function name for `vuLogSetFile()`.

Preferred function name:

- [vuLogSetFile()](vuLogSetFile.md)

It behaves the same way as `vuLogSetFile()` and changes the current runtime log file path.
