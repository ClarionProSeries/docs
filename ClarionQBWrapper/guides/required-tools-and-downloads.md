# Required Tools and Downloads

The following tools are required or strongly recommended when working with the Clarion ProSeries QuickBooks Wrapper and QuickBooks Desktop.

---

## Required

- QuickBooks Desktop (Pro, Premier, or Enterprise edition)
- QuickBooks SDK (QBXML SDK)
- A valid developer license key (provided by LANSRAD)

## Strongly recommended

- DebugView by Sysinternals or DebugView++ for viewing wrapper debug output
- Visual Studio if you licensed the source code and plan to rebuild the C# DLL

---

- Request a Free Developer License (NFR)

- Download DebugView  
  https://learn.microsoft.com/en-us/sysinternals/downloads/debugview

---

## Included with your license or installer

All required wrapper components are provided at licensing time or installed by the product installer:
- Clarion template
- LIB
- DLL
- Classes
